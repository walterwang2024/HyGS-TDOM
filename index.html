HyGS-TDOM

Hybrid Gaussian Splatting for True Digital Orthophoto Map Generation
This project proposes a hybrid framework combining 3DGS and FSGS to address sparse-view TDOM challenges, achieving high-quality true orthophotos from UAV imagery.
